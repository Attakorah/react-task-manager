# React Task Manager

A modern task manager application built with React and Vite.

This project is part of my journey toward becoming a Full-Stack AI Developer. It helped me practice React state management, forms, events, conditional rendering, local storage, and filtering.

## Features

- Add new tasks
- Mark tasks as completed
- Undo completed tasks
- Delete tasks
- Clear all completed tasks
- Filter tasks by:
  - All
  - Active
  - Completed
- Completed task counter
- Local storage persistence
- Responsive modern UI

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- Git
- GitHub

## React Concepts Practiced

- `useState`
- `useEffect`
- Controlled inputs
- Form handling
- Event handling
- Rendering lists with `.map()`
- Updating arrays in state
- Filtering arrays with `.filter()`
- Conditional rendering

Installation and Setup

Clone the repository:

git clone https://github.com/Attakorah/react-task-manager.git

Go into the project folder:

cd react-task-manager

Install dependencies:

npm install

Run the development server:

npm run dev

Then open the local URL shown in your terminal, usually:

http://localhost:5173/
Build for Production
npm run build
